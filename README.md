**Objetivo General:**
El objetivo del curso es dotar al alumno de las herramientas teóricas y prácticas necesarias para comprender y aplicar técnicas de estimación que combinan Econometría con Ciencia de Datos (Machine Learning) en el análisis de la relación entre variables económicas. 

**Objetivos Específicos:**
- Aprender de manera práctica las coincidencias y diferencias entre econometría y ciencia de datos.
-	Aprender la teoría estadística y econométrica que sustenta modelos que combinan econometría y ciencia de datos.
-	Aprender y aplicar herramientas complementarias a econometría: ridge regression, Lasso, elastic net, regression trees y random forests.
-	Interpretar resultados de investigación que emplean econometría aplicada y machine learning.  
-	Desarrollar habilidades para la extracción, administración, procesamiento, modelación e interpretación de datos económicos.
